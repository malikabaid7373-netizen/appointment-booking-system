# Appointment Booking System

A bilingual appointment booking system for clinics in Saudi Arabia.

## Project Goal

The goal of this project is to help clinics manage doctors, services, customers, and appointments through a responsive web application.

## Target Users

- Clinic administrators
- Reception staff
- Doctors
- Customers

## MVP Features

### Public Website

- Arabic and English languages
- Responsive landing page
- Clinic services
- Doctors listing
- Doctor details
- Appointment booking form
- Contact information

### Customer Features

- Sign up and login
- View doctors and services
- Book an appointment
- View personal appointments
- Cancel an appointment
- Update profile

### Admin Features

- Admin login
- Dashboard statistics
- Manage doctors
- Manage services
- View appointments
- Update appointment status
- Manage customers

## Technology Stack

- Next.js
- TypeScript
- Tailwind CSS
- Supabase
- PostgreSQL
- Git and GitHub

## Initial Project Scope

This first version will focus only on clinics.

Salons, gyms, and other businesses may be supported in future versions.
